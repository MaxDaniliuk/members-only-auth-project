# Members Only Authentication Project

A full-stack web application that demonstrates user authentication, access control, and message posting. Logged-in users can post messages, but only authorized users can view post details like post author and date of post creation, creating a "members-only" experience. There is also an admin functionality for deleting posts. However, admin has to be created in advance.
